# OTP Verification

A simple web application for OTP (One-Time Password) verification.

## Project Structure

- `index.html`: The main entry point for the OTP login/verification interface.
- `sample-config.js`: Configuration template for setting up external services (like Firebase or similar).

## Getting Started

1.  **Configure Environment**: Rename `sample-config.js` to `config.js` and fill in your specific credentials.
2.  **Run Locally**: Open `index.html` in your browser to view the interface.

## Purpose
This project provides a clean, reusable UI component for implementing security flows requiring OTP validation.
